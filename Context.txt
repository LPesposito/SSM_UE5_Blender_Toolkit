📄 Contexto de Manutenção: Scene StaticMesh - UE5 Workflow Toolkit
Descrição do Projeto:
Este é um Addon para Blender (Python) focado na exportação de cenas completas ou Static Meshes individuais para a Unreal Engine 5. Originalmente concebido para migração de projetos legados (BGE), evoluiu para uma ferramenta robusta de organização de cena e exportação automatizada.
Principais Funcionalidades:
Visualização: Toggle de Backface Culling para identificar faces invertidas.
Geometria: Aplicação de Transforms (Scale/Rotation) e ajuste de Pivot Point em 6 direções (Bottom, Center, Left, Right, Front, Back) usando cálculos de matriz (mathutils).
Colisões: Geração automatizada de colisões UCX (Convex Hull via bmesh + Decimate 50%), com sistema de parentesco automático (offset zero via inversão de matriz) e organização em coleção dedicada ("Collisions").
Materiais: Suporte a múltiplos slots de material por objeto. Renomeação inteligente (M_Objeto_Material) e texturas (T_Material_Sufixo) com análise de nós Principled BSDF e prevenção de duplicatas via set de materiais processados.
Exportação: Exportação FBX otimizada (eixos -Z Forward, Y Up). Suporte para exportação individual com correção automática de pivô (move temporariamente para 0,0,0 e aplica bake) e inclusão automática de colisores filhos (UCX, UBX, USP, UCP). Conversão de intensidade de luz integrada.
Arquitetura do Código:
Estrutura baseada em bpy.types.Operator, bpy.types.Panel e bpy.types.PropertyGroup.
Uso de bmesh para operações de geometria e PointerProperty para persistência de dados.
Lógica de Undo (UNDO flag) habilitada em todos os operadores.
Sistema de busca recursiva de nós de imagem (find_image_node) para ignorar nós intermediários (Normal Maps, Ramps, etc.).
Separação modular em __init__.py, operators.py, properties.py, ui.py e utils.py.
